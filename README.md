# CIFAR-10 Görüntü Sınıflandırma (CNN)

Bu projede, CIFAR-10 veri seti kullanılarak bir Convolutional Neural Network (CNN) modeli ile görüntü sınıflandırma işlemi gerçekleştirilmiştir.
## 🚀 Proje Özeti

* TensorFlow/Keras kullanılarak CNN modeli geliştirildi
* CIFAR-10 veri setindeki 10 farklı sınıf üzerinde sınıflandırma yapıldı
* Modelin eğitimi, doğrulanması ve performans değerlendirmesi gerçekleştirildi

## 🧠 Model Mimarisi

* Convolutional katmanlar (Conv2D + ReLU)
* MaxPooling katmanları
* Dropout (overfitting’i azaltmak için)
* Fully Connected (Dense) katmanlar
* Softmax çıkış katmanı


## 🛠 Kullanılan Teknolojiler

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib


## 📂 Proje Yapısı

* `cifar10_cnn.ipynb` → Modelin eğitildiği ve test edildiği notebook
* `README.md` → Proje açıklamaları


## 🎯 Kazanımlar

* CNN mimarisi ve görüntü sınıflandırma mantığını öğrenme
* Model eğitimi ve performans değerlendirme süreçlerini anlama
* Overfitting problemini dropout ile azaltma
* Gerçek veri setleri ile çalışma deneyimi

## 💡 Geliştirme Fikirleri

* Hyperparameter tuning
* Data augmentation
* Transfer learning (ResNet, VGG vb.)
* Model performansını artırma çalışmaları
